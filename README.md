# Mini Avito

Веб-приложение на Flask для размещения объявлений о продаже товаров.  
Пользователи могут регистрироваться, входить в аккаунт, публиковать товары, редактировать и удалять их, а также общаться через встроенные чаты.

## Возможности

- регистрация и авторизация пользователей
- личный кабинет
- создание, редактирование и удаление объявлений
- загрузка изображений товаров
- просмотр карточки товара
- фильтрация объявлений по городу и категории
- чаты между продавцом и покупателем
- список всех чатов пользователя
- отображение непрочитанных сообщений

## Установка и запуск
### 1. Клонировать репозиторий
``` 
git clone https://github.com/angywwx/mini_avito.git
cd mini_avito
```
### 2. Создать виртуальное окружение
``` 
python -m venv .venv
source .venv/bin/activate
```
### 3. Установить зависимости и tuna
``` 
pip install -r requirements.txt
sh -c "$(curl -sSLf https://releases.tuna.am/tuna/get.sh)" 
tuna config save-token <your_token>
```
### 4. Запустить проект
``` 
python main.py
tuna http 5000
``` 
## Классы

### User
Пользователь системы:

- имя
- фамилия
- город
- email
- пароль

### Goods
Объявление:

- название
- описание
- цена
- изображение
- категория
- город
- дата создания
- владелец

### Category
Категория товара:

- одежда
- обувь
- электроника
- книги
- и др.

### Chats
Чат между продавцом и покупателем по конкретному товару.

### Messages
Сообщения внутри чата.

## Функционал пользователя

После регистрации пользователь может:

- войти в аккаунт
- открыть личный кабинет
- создать объявление
- загрузить фотографию товара
- просматривать свои объявления
- редактировать и удалять свои объявления
- просматривать карточки товаров других пользователей
- писать продавцам
- просматривать список своих чатов

## Чаты

Для каждого товара можно открыть чат между продавцом и покупателем.

В чате:

- свои сообщения отображаются справа
- сообщения собеседника отображаются слева
- список чатов сортируется по последнему сообщению
